# ความเข้าใจระบบการเงิน โรงเรียนวัดบ้านดอน

> บันทึกความเข้าใจจากการวิเคราะห์ข้อมูล Excel และพูดคุยกับผู้ใช้งาน
> สถานะ: กำลังรวบรวมข้อมูล (ยังไม่ลงมือออกแบบระบบ)

---

## 1. ภาพรวม Workflow

```
Flow การทำงานของครูการเงิน (7 ขั้นตอน)

1. ยื่นเรื่องของบ (ด้วยวาจา)
2. ทำหนังสือขออนุมัติเบิกงบ → พิมพ์บันทึกขออนุมัติ
3. ไปเบิกเงินที่ธนาคาร
4. นำจ่ายผู้รับจ้าง
5. ออกใบ 50 ทวิ ให้ผู้รับจ้าง
6. หนังสือแจ้งยอดเงินคงเหลือ → พิมพ์รายงานคงเหลือ
7. ทำบันทึกสรุปภาษี
```

---

## 2. เอกสารหลัก 2 ฉบับ

### 2.1 บันทึกขออนุมัติ (เงินออก เท่านั้น)

- 1 ฉบับ = 1 วันที่ = หลายรายการ = หลายบัญชี/ประเภทเงินได้
- รายการจัดกลุ่มตามประเภทเงิน
- มียอดรวมแต่ละประเภท + ยอดรวมทั้งหมด
- ต้องลงนาม: ผู้ขอเบิก → รอง ผอ. → ผอ.
- ตัวอย่าง: 1 ฉบับ อาจมีทั้งค่าจ้างแม่ครัว (อาหารกลางวัน) + ค่าวัสดุ (อุดหนุนทั่วไป)

### 2.2 รายงานเงินคงเหลือ (ทั้งเงินเข้า + เงินออก)

- สร้างได้ทุกวันที่มีเงินเคลื่อนไหว
- แสดงยอดสะสมทุกประเภทบัญชี ณ วันนั้น
- **ไม่ได้ผูกกับบันทึกขออนุมัติเสมอไป**

| เหตุการณ์ | บันทึกขออนุมัติ | รายงานคงเหลือ |
|----------|:-:|:-:|
| เงินอุดหนุนโอนเข้าบัญชี | ❌ ไม่มี | ✅ ต้องทำ (ยอดเพิ่ม) |
| เบิกเงินจ่ายผู้รับจ้าง | ✅ ต้องทำ | ✅ ต้องทำ (ยอดลด) |
| สิ้นเดือน ตรวจยอด | ❌ ไม่มี | ✅ ทำได้ (snapshot) |

## 3. โครงสร้างข้อมูลที่เข้าใจ

### 3.1 บัญชีธนาคาร (BankAccount)
- ชื่อธนาคาร, เลขบัญชี, ชื่อบัญชี, สาขา
- ประเภทบัญชี (ออมทรัพย์/กระแสรายวัน)
- ผูกกับประเภทเงิน (BudgetType)
- สถานะ (ใช้งาน/ไม่ใช้งาน)

### 3.2 ประเภทเงิน (BudgetType) — จากข้อมูล Excel
- เงินอุดหนุนทั่วไป
- เงินอุดหนุนโครงการอาหารกลางวัน
- เงินอุดหนุนเทศบาล
- เงินเรียนฟรี 15 ปี (มี 4 ย่อย: ค่าอุปกรณ์, ค่าเครื่องแบบ, ค่ากิจกรรม, ค่าหนังสือ)
- เงิน กสศ. (ทุนเสมอภาค)
- เงินรายได้สถานศึกษา
- เงินประกันสัญญา
- เงินหักภาษี ณ ที่จ่าย

### 3.3 ประเภทบัญชีในรายงานคงเหลือ — จาก Excel (11+ รายการ)
- ดอกเบี้ยเงินฝากเงินอุดหนุนทั่วไป
- เงินอุดหนุนค่าใช้จ่ายในการจัดการศึกษา
- เงินเรียนฟรี 15 ปี (4 ย่อย)
- เงิน กสศ.
- เงินโครงการอาหารกลางวัน
- เงินโครงการจ้างครูฯ
- เงินรายได้สถานศึกษา (2 ย่อย)
- เงินประกันสัญญา (หลายรายการ)

### 3.4 บันทึกขออนุมัติ (ApprovalRequest)
- วันที่, เลขฉบับ
- มีหลายรายการ (DisbursementItem)
- แต่ละรายการ: ประเภทเงิน + บัญชีธนาคาร + รายการ + จำนวนเงิน + หักภาษี
- สถานะ workflow (7 ขั้นตอน)

### 3.5 รายการเคลื่อนไหวเงิน (MoneyTransaction)
- ประเภท: เงินเข้า (IN) หรือ เงินออก (OUT)
- บัญชีธนาคาร + ประเภทเงิน + จำนวนเงิน
- ถ้าเป็นเงินออก → ผูกกับบันทึกขออนุมัติ
- ถ้าเป็นเงินเข้า → ระบุที่มา (เช่น "รับเงินอุดหนุนจากเทศบาล")

### 3.6 รายงานเงินคงเหลือ (BalanceReport)
- คำนวณจาก MoneyTransaction สะสมทั้งปีงบประมาณ
- แสดงยอดแต่ละประเภทบัญชี: เงินสด, ฝากธนาคาร, ส่วนราชการผู้เบิก, รวม
- สร้างได้อิสระ ไม่จำเป็นต้องผูกกับบันทึกขออนุมัติ

## 4. บุคลากร (จากข้อมูล Excel)

| ชื่อ | ตำแหน่ง | บทบาทในระบบ |
|------|---------|------------|
| นางมณฑิรา สายยศ | ครูชำนาญการพิเศษ | เจ้าหน้าที่การเงิน (กรอกข้อมูล, สร้างเอกสาร) |
| นางสาวสุทธิดา สุทธิ | รองผู้อำนวยการ | ตรวจสอบ/อนุมัติ |
| นางภควรรณ มีเจริญ | รองผู้อำนวยการ | ตรวจสอบ/อนุมัติ |
| นางสาววิภาพรรณ อุบล | ผู้อำนวยการ | อนุมัติ |

## 5. Roles ในระบบ (5 roles)

| Role | Code | สิทธิ์หลัก |
|------|------|-----------|
| ครู | TEACHER | ดูข้อมูลที่เกี่ยวข้อง |
| ครูการเงิน | FINANCE_OFFICER | กรอก/สร้าง/ดำเนินการ workflow |
| รองผู้อำนวยการ | VICE_PRINCIPAL | ตรวจสอบ/อนุมัติ |
| ผู้อำนวยการ | PRINCIPAL | อนุมัติ/ดูรายงาน |
| ผู้ดูแลระบบ | ADMIN | จัดการผู้ใช้/สิทธิ์/ระบบ |

Admin สามารถปรับ permission ของแต่ละ role ได้ (Custom permission)

## 6. เอกสารราชการ — รูปแบบการพิมพ์

### ตราครุฑ
- ทุกเอกสารราชการต้องมี **ตราครุฑ** ด้านบนกลางหน้า
- ไฟล์ตราครุฑ: `public/garuda.png`
- ขนาดบนเอกสาร: ประมาณ 60-80px (ตามมาตรฐานเอกสารราชการ)
- ใช้ในเอกสาร:
  - บันทึกข้อความ (บันทึกขออนุมัติเบิกงบ)
  - หนังสือราชการอื่นๆ ที่พิมพ์จากระบบ

### รูปแบบหัวเอกสาร "บันทึกข้อความ"
```
            [ตราครุฑ]
          บันทึกข้อความ

ส่วนราชการ  โรงเรียนวัดบ้านดอน สพป.ระยอง เขต 1
ที่  _________          วันที่  _______________
เรื่อง  ขออนุมัติเบิกเงินอุดหนุน
```

## 7. UI/UX ที่ตกลงแล้ว

- Theme: น้ำเงินเข้ม (#1e3a5f) + เขียว (#16a34a) + ขาว (Light theme)
- Icon: Lucide React เท่านั้น (ห้าม emoji)
- Font: Sarabun + IBM Plex Mono (ตัวเลข)
- UI Library: Tailwind CSS + shadcn/ui
- Sidebar: ย่อ/ขยายได้ (64px / 240px) พื้นน้ำเงินเข้ม
- Header: Fixed top + Tab Modules (เพิ่มโมดูลได้ เช่น นักเรียน, บุคลากร)
- Login: Password (Desktop) + LINE (มือถือ — ดู + อนุมัติ)
- Mobile: Bottom Navigation + Drawer sidebar
- Print: A4, font Sarabun, CSS @media print
- Deploy: Railway (Next.js + PostgreSQL)
- Local Dev: Docker Compose
